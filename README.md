## Milestone 2 (lexer)

There are 6 testcases in  `./tests/lexer/`. To run test `n`, run the following command in the main (`compiler7`) directory: 

```console
make n
```
(`n = 1,2,...,6`)

Table with tokens, lexemes, line# and column# will be displayed.
